# 🎮 DooR DasH – Scare vs Laugh Touchdown

DooR DasH is a **strategy-based board game implemented in Java using JavaFX**, inspired by the world of *Monsters Inc.*.

Players compete on a **100-cell zigzag board** collecting energy from doors while interacting with dynamic game mechanics such as special cells, cards, and monster abilities. The first player to reach the final cell with sufficient energy wins the game.

The project focuses on **object-oriented design, modular architecture, and complex game state management**.

---

# 🧠 Gameplay Overview

The board contains multiple interactive elements that influence gameplay:

- **Door Cells** – Energy gain or loss depending on the monster role  
- **Transport Cells** – Move players forward or backward (similar to Snakes & Ladders)  
- **Card Cells** – Trigger random events such as energy steal, shields, or role confusion  
- **Monster Cells** – Activate special abilities  
- **Normal Cells** – Standard movement spaces  

Players must balance **movement, energy management, and strategic ability usage** to reach the final cell with **≥1000 energy**.

# 🏗️ Technical Architecture

The project was designed using **object-oriented software engineering principles** with clear separation between game logic and user interface.

### Key Components

**Game Engine**
- Manages turns, dice rolling, and win conditions
- Handles overall game state and rule enforcement

**Board System**
- Represents the 100-cell grid
- Handles cell interactions and movement logic

**Monster System**
- Defines monster roles, abilities, and energy mechanics
- Uses inheritance and polymorphism to support different monster types

**Cell System**
- Implements different board cell behaviors (door, card, transport, monster)

**Card Engine**
- Manages card deck, drawing, and effects

**JavaFX UI**
- Visualizes the board and player movement
- Displays game state, energy levels, and turn updates
